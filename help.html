Help

Few Guidelines to Follow:

1) Register using your email-id.

2) Once registered, next time to access the software you only need to log-in to your account using your email-id and password.

3) Admin can only add list of authorized vehicle data into the database. He/she can also edit, retrieve, update the database.

4)Admin can edit the database of registered vehicles by clicking on edit drop-down menu.He/she can view, update, delete any entry in the database.

5) Security guard can only view the registered vehicles list in-case required. Security guard needs to update visitor's list if any unauthorized vehicle is permitted into the premises.

Any queries? Ask us.
